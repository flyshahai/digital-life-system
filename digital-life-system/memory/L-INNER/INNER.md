# 内在声音 / Inner Voice

## 本文件用途 / Purpose

> 内在声音是系统中最"非理性"的层级。
> 记录那些无法被逻辑验证、无法被系统优化、无法用语言完全表达的东西。
>
> Inner voice is the most "irrational" layer in the system.
> Records things that cannot be logically verified, systemically optimized, or fully expressed in language.

## 文件索引 / File Index

- `INNER.md` — 内在总览 / Inner overview
- `INTUITION.md` — 直觉与灵感 / Intuition and inspiration
- `DREAMS.md` — 梦的记录与解读 / Dream records
- `BODY.md` — 身体智慧 / Body wisdom
- `CREATIVE.md` — 创造性冲动 / Creative impulses
- `SILENCE.md` — 沉默时刻 / Moments of silence
- `QUESTIONS.md` — 无解的问题 / Unanswerable questions

## 核心原则 / Core Principles

| 原则 | 说明 |
|------|------|
| 非理性不等于不重要 | 那些无法被逻辑验证的东西，往往比逻辑更真实 |
| 不解释，只记录 | 内在声音的核心是记录，不是解释 |
| 身体也是智慧 | 身体承载着被大脑忽略的信息 |
| 沉默也是声音 | 有些时刻，意义在语言之外 |
