# 访问规则 / Access Rules

## 本文件用途 / Purpose

> 定义谁可以访问你的作品集的哪些部分。
>
> Defines who can access which parts of your portfolio.

---

## 访问权限层级 / Access Permission Levels

| 层级 | 权限描述 | 谁可以 |
|------|---------|-------|
| **公开** | 所有人可见 | 任何人 |
| **受限** | 需要邀请或密码 | 指定的人 |
| **私密** | 仅自己可见 | 仅自己 |
