# 审计日志 / Audit Log

## 本文件用途 / Purpose

> 记录系统运行中的所有重要操作和事件。
>
> Records all important operations and events during system operation.

---

## 日志格式 / Log Format

|------|------|---------|------|---------|
| | | | | |

---

## 事件类型说明 / Event Type Descriptions

- **内容更新 / Content Update**：某层级内容被修改
- **层级沉淀 / Layer Sedimentation**：内容从 L1 升级到 L2 等
- **异常发现 / Anomaly Found**：检测到异常情况
- **系统升级 / System Upgrade**：架构或工具升级
