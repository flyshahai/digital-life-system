## 选择你的路径 / Choose Your Path

### 🟢 第一次使用 / First Time

如果你刚刚下载了这个系统，建议按以下顺序开始：

1. **阅读 SYSTEM.md** — 理解系统的运行逻辑
2. **从 L4 IDENTITY.md 开始** — 这是系统的校准锚点
3. **填写 L5 ANCHOR.md** — 写下你的灵魂宣言
4. **按需扩展** — 添加你需要的扩展模块

### 🟡 有经验的用户 / Experienced Users

如果你已经有自我认知系统，可以：

1. 将现有内容迁移到对应模块
2. 检查各层内容的一致性
3. 补充缺失的层级

### 🔵 想深入探索 / Deep Exploration

如果你想进行深度自我探索：

1. **从 L-SHADOW 开始** — 直面自己的阴影
2. **L-INNER** — 倾听内在声音
3. **L5 ANCHOR.md** — 写下你的灵魂宣言
