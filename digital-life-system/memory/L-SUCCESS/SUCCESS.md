# 成功引擎 / Success Engine

## 本文件用途 / Purpose

> 成功引擎是系统的"执行层"。
> 把愿景转化为行动，把经验转化为可复制的成功模式。
>
> Success engine is the "execution layer" of the system.
> Transforms vision into action, experience into replicable success patterns.

## 文件索引 / File Index

- `SUCCESS.md` — 成功引擎总览 / Success engine overview
- `STRATEGY.md` — 战略导航（目标分解与路径规划）/ Strategic navigation
- `ANCHORS.md` — 行为锚定（关键时刻的行为模式）/ Behavioral anchors
- `PATTERNS.md` — 成功模式（可复制的成功经验）/ Success patterns
- `TRACKING.md` — 成就追踪（里程碑与进度）/ Achievement tracking
- `FAILURE.md` — 失败学习（从失败中提取教训）/ Failure lessons

## 核心原则 / Core Principles

| 原则 | 说明 |
|------|------|
| 行动优先 | 完美的计划如果不去执行，也只是一堆文字 |
| 迭代改进 | 失败不是终点，而是学习的机会 |
| 环境设计 | 意志力是有限的，但环境设计可以减少对意志力的依赖 |
| 能量管理 | 高效不是做更多，而是做最重要的事 |
