## 🚀 快速开始 / Quick Start

### 方式一：手动安装 / Manual Install

1. **克隆仓库 / Clone the repo**
   ```bash
   git clone https://github.com/shahai/digital-life-system.git
   cd digital-life-system
   ```

2. **选择安装层级 / Choose your tier**
   - 🟢 核心包：直接使用 `memory/` 目录
   - 🟡 推荐扩展：额外添加 L-SUCCESS / L-LEARNING / L-relations / L-PROJECTS
   - 🔵 深度模块：按需添加

3. **将 memory/ 目录放入你的 AI 工具工作区 / Integrate with your AI**
   - WorkBuddy：放入 `.workbuddy/memory/`
   - Cursor：放入项目 `.cursor/memory/`
   - Claude.ai / ChatGPT：将 `memory/` 路径告知 AI

4. **让 AI 读取 SYSTEM.md / Let AI read SYSTEM.md**
   > 将 `SYSTEM.md` 的内容作为 system prompt 或告诉 AI："请阅读 `memory/SYSTEM.md`，这是我的个人认知操作系统。"

### 方式二：WorkBuddy 一键安装 / WorkBuddy One-Click Install

> 适用于 WorkBuddy 用户

1. 在 WorkBuddy 中导入 Skill
2. 系统自动初始化 `memory/` 目录
3. 开始填写你的内容

## 📦 模块说明 / Modules

### 核心包（必装）/ Core Package

| 模块 | 说明 | Module |
|------|------|--------|
| `L0` | 工作记忆，会话级临时存储 | Working memory, session-level temporary |
| `L1` | 情境层，近期上下文 | Situational layer, recent context |
| `L2` | 行为层，习惯与偏好 | Behavioral layer, habits and preferences |
| `L3` | 认知层，思维模式与决策框架 | Cognitive layer, thinking patterns |
| `L4` | 人格层，核心价值观 | Persona layer, core values |
| `L5` | 灵魂层，精神内核与意志声明 | Soul layer, spiritual core |

| 模块 | 说明 | 模块类型 |

| `L-CREATIONS` | 创作层：作品、知识贡献 | 🔵 深度 |
