# 项目层 / Projects Layer

## 本文件用途 / Purpose

> 项目层追踪长期项目和使命的进展。
>
> Projects layer tracks progress on long-term projects and missions.

## 文件索引 / File Index

- `PROJECTS.md` — 项目总览 / Projects overview
- `CALLING.md` — 使命宣言 / Calling statement
- `ACTIVE/` — 进行中的项目 / Active projects
- `ARCHIVED/` — 已完成/暂停的项目 / Archived projects

---

## 项目分类 / Project Categories

| 类别 | 说明 |
|------|------|
| **使命级** | 人生级别的项目，可能持续数年 |
| **项目级** | 有明确目标和期限的具体项目 |
| **实验级** | 小规模的尝试和探索 |

---

*维护节奏：每周更新活跃项目状态，每项目标完成后记录到归档。*
*Maintenance rhythm: Update active project status weekly, archive upon completion.*
