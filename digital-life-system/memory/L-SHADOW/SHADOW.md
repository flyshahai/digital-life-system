# 阴影档案 / Shadow Archive

## 本文件用途 / Purpose

> 阴影档案记录那些被否认、被压抑、被隐藏的自我面向。
> 这不是羞耻的记录，而是诚实的档案。
>
> Shadow archive records self-aspects that are denied, repressed, or hidden.
> This is not a record of shame, but a record of honesty.

## 文件索引 / File Index

- `SHADOW.md` — 阴影总览与哲学 / Shadow overview and philosophy
- `DARK-TRAITS.md` — 不被承认的特质 / Unacknowledged traits
- `REPRESSED.md` — 被压抑的欲望与恐惧 / Repressed desires and fears
- `PROJECTIONS.md` — 投射出去的特质 / Projected traits
- `UNCLAIMED.md` — 被遗弃的梦想 / Abandoned dreams
- `INTEGRATION.md` — 阴影整合记录 / Shadow integration records

## 核心原则 / Core Principles

| 原则 / Principle | 说明 / Description |
|----------------|-------------------|
| 非评判 / Non-judgment | 阴影档案不是道德审判的场所 / Shadow archive is not a place for moral judgment |
| 诚实 / Honesty | 记录阴影需要勇气，系统承诺这些内容不会被用于评判 / Recording shadows takes courage; system promises this content will not be used for judgment |
| 整合 / Integration | 记录阴影不是为了沉溺，而是为了整合 / Recording shadows is not for wallowing, but for integration |

## 默认选择 / Default Choice

> 死亡协议中，默认选择"完整保留"阴影档案。
> 一个没有阴影的数字身份，不是完整的你。
>
> In the death protocol, the default choice is "complete preservation" of the shadow archive.
> A digital identity without shadows is not the complete you.
